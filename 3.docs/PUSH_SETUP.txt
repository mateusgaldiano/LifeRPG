COMO FUNCIONA O AGENDAMENTO
------------------------------------------------------------------------
O pg_cron chama a Edge Function todo dia às 22:00 UTC (= 19:00 Brasília)
com {"action":"trigger_all_reminders"}. A função busca quem tem missões
pendentes (quests.completed = false) e manda o lembrete só pra essas
pessoas. Para mudar o horário, edite o schedule no SQL (em UTC).

OBS: os horários personalizados do app (slider Manhã/Noite) ainda NÃO
são usados pelo servidor — isso é fase 2 (precisa sincronizar o fuso de
cada usuário pro banco). Hoje o lembrete é em horário fixo pra todos.
