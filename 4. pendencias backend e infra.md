## 🔴 Crítico

### 1. Criar Projeto no Firebase e Configurar Chaves (Cloud Save)
O código fonte (UI, Lógica de Sync, Fallback Offline) já está **100% implementado** no código do The System v2.0. Falta apenas o usuário:
1. Criar um projeto gratuito no [Firebase Console](https://console.firebase.google.com/).
2. Habilitar o **Firestore Database** e o **Authentication** (Google Provider).
3. Copiar as chaves geradas e colar no arquivo `firebase-config.js` (descomentando a última linha).

**Bloqueia:** Todos os itens sociais (Ranking, PvP) e as Push Notifications nativas.

## 🟡 Médio
