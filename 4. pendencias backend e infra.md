## 🔴 Crítico

### 1. Cloud Save (Sincronização entre Dispositivos)
Atualmente tudo vive no `localStorage` do navegador — sujeito a limpeza de cache, perda em troca de aparelho e sem sync entre PC e celular.

**Implementação:** Firebase Firestore + Auth. O `gameState` completo vai para a nuvem a cada `saveGameData()`. Login por Google ou e-mail.

**Bloqueia:** Todos os itens sociais (Ranking, PvP).

---

## 🟡 Médio

### 2. Push Notifications Reais (Nativas do OS)
Lembretes enviados mesmo com PWA fechado. A UI de horários já existe em Configurações (`morningHour`, `eveningHour`). Falta o disparo real via Service Worker + Web Push API.

**Implementação esperada:** Firebase Cloud Messaging (FCM) ou Web Push API com servidor backend mínimo para guardar as chaves VAPID dos dispositivos.

**Esforço:** Médio/Alto (requer servidor para gerir subscriptions).
