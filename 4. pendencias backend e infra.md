# Pendências de Backend e Infraestrutura — LifeRPG OS v2.0

Esta lista abrange toda a infraestrutura técnica necessária para suportar funções de nuvem, persistência remota e notificações no nível do sistema operacional (OS).

## 🔔 Push Notifications Reais
- **Objetivo:** Enviar lembretes e notificações para o celular do usuário, mesmo quando o aplicativo (PWA) estiver fechado.
- **Implementação esperada:** Integração via Firebase Cloud Messaging (FCM) ou Web Push API nativa usando um servidor backend. Atualmente as notificações dependem de timers locais ou da permissão do Notification API no client-side.

## ☁️ Cloud Save (Sincronização entre Dispositivos)
- **Objetivo:** Permitir que o jogador acesse sua conta do celular e do computador de forma sincronizada, substituindo a persistência puramente baseada em `localStorage`.
- **Implementação esperada:** Integração com Firebase Firestore / Auth para armazenar o `gameState` do usuário em tempo real na nuvem.
