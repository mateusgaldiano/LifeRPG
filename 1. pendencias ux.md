## 🔴 Crítico

### 1. Títulos do Avatar Zoom desalinhados com Ranks
`openAvatarZoom()` usa thresholds fixos (LV5/10/15/20) que não batem com o sistema de rank real (E/D/C/B/A/S até LV30). RANK S (LV30+) não tem título.

**Fix:** Alinhar via `getRankForLevel()`, mapeando título por rank: E → "Recruta", D → "Aventureiro", C → "Caçador", B → "Elite", A → "Herói Lendário", S → "O Sistema".

---

### 2. Bug: Dungeon ativa + "NENHUMA MISSÃO ATIVA" simultâneos
Condição do fallback não considera a dungeon ativa.

**Fix:**
```javascript
if (gameState.sideQuests.length === 0 && !gameState.activeDungeon) {
    sideContainer.innerHTML = `<div ...>NENHUMA MISSÃO ATIVA</div>`;
}
```

## 🟡 Médio

### 3. Perks e Sinergias sem separação visual
Para jogadores de LV15+ com múltiplas sinergias, os badges viram uma massa sem hierarquia.

**Fix:** Labels separadores "PERKS" e "SINERGIAS" acima de cada grupo, ou cor de borda diferente (cyan para sinergias, dourado para perks).
