## ✅ Resolvidos nesta sessão

| Item | O que foi feito |
|------|----------------|
| ~~`alert()` nativo (Export/Import/Reset)~~ | Substituído por `showSystemToast()` em todo o código |
| ~~Boss Quest sem UI persistente~~ | Weekly Boss card vermelho pulsante + modal de chegada implementados |
| ~~Dungeon sem destaque visual~~ | Banner standalone roxo-neon com timer, tag de skill e botão dedicado |
| ~~Streak overflow no HUD mobile~~ | CSS `white-space: nowrap` + layout compacto no player-info |
| ~~Barra de progresso para água~~ | Mini barra proporcional no card de água (q-agua) |
| ~~Indicador persistente de dia completado~~ | Classe `.all-complete` com badge dourado no header das dailies |
| ~~Cabeçalho fixo mobile~~ | Scroll movido para body; `.player-info` com `position: sticky` corrigido |
| ~~Inventário / Taverna sem sub-abas~~ | Sub-abas Loja / Inventário implementadas com `switchTavernaTab()` |
| ~~Conquistas sem UI~~ | Sala de Troféus com 14 conquistas, categorias, raridade e barras de progresso |
| ~~Títulos do Avatar Zoom desalinhados~~ | Alinhados com `getRankForLevel()` usando `titleMap` mapeado por rank |
| ~~Bug Dungeon + "Nenhuma missão"~~ | Corrigida a condição de renderização para considerar se há dungeon ativa |
| ~~Perks e Sinergias sem separação visual~~ | Adicionados títulos e linhas separadoras coloridas (`#fbbf24` / `#00f0ff`) |
